# BabyFST
Babylonian Finite-State Morphology v. 2.0

# Requirements
For using the transducer you need Foma https://fomafst.github.io/. For evaluation you will need the HFST PyPi https://pypi.org/project/hfst/.

# Recompile
Use files in `src` folder.
